<div align="center">
  <h1>Apple Music Electron</h1>
  An open-source, GPU-accelerated Electron application that presents the Apple Music website in a customizable interface.
  <br/>
  It is intended to be used as a native Apple Music app on platforms that don't have one (Linux and Windows).
  <br/>
  This is a fork and successor of <a target="_blank" rel="noopener" href="https://github.com/ciderapp/Apple-Music-Electron">the original</a>.
</div>
